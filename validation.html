Validation

Les resources FHIR peuvent être validées via l'outil Validator CLI. Actuellement, la validation des resources sur un serveur FHIR n'est pas supportée (opération $validate)

Comment valider les resources ?

Installer dotNET SDK 8 winget install Microsoft.DotNet.SDK.8

Installer Firely terminal dotnet tool install -g firely.terminal

Installer Java JRE 17+

Télécharger validator CLI here

Charger les guides non publiés dans le cache FHIR local:

  • Télécharger EU HDR
  • Installer EU HDR: fhir install euhdr-0.1.0-ballot.tgz --file
  • Télécharger le package de cet IG ici et l'installer: fhir install package.tgz --file.

Exécuter le validateur sur vos resources. Par exemple: java -jar ..\fhir-validator\validator_cli.jar output\Bundle-ChulPneumoBundleExample.json -ig be.chuliege.versunsoufflenouveau

Examiner la sortie du validateur. Voir documentation
